Aionoscope Foundational Benchmark Dashboard

Single-page ECharts view over the frozen-feature offline-probe JSON outputs in results/models/. The page computes the same 6 radar panels and 4 layer curves in browser-side JavaScript, without calling the Python plotting code. When JSONs contain repeated validation runs, the charts use median ± std across the validation-seed pool.

Negative R2 values are clipped to 0 in the radar and layer-curve plots only, matching the benchmark visualization convention. Tooltips still show the raw unclipped R2 median together with its standard deviation. Standard deviation uses sample ddof=1 when multiple validation runs are present. For automatic JSON discovery, serve this repo with a static file server so the browser can fetch results/models/.
